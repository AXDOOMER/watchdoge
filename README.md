# watchdoge
Port scanner for vulnerabilities

### Usage

```
> go build watchdog.go 
> ./watchdog 192.168.1.1
tcp port 21 open
tcp port 22 open
tcp port 53 open
tcp port 80 open
tcp port 2000 open
tcp port 8291 open
tcp port 8728 open
tcp port 8729 open
done
```
